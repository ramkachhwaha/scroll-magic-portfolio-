A visually stunning, interactive 3D portfolio website built to showcase full-stack development skills and creative web design. This project uses 3D elements to create an immersive user experience that stands out from traditional 2D portfolios.

---

##  What the project does

This website serves as a digital resume and business card for a **MERN Stack Developer**. It allows users to:
* **Explore 3D Scenes:** Navigate through an interactive 3D environment that changes as you scroll.
* **View Projects:** A dedicated section showcasing full-stack applications like E-commerce and Chat apps.
* **Technical Breakdown:** Displays proficiency in Frontend (React, Tailwind), Backend (Node, Express), and Databases (MongoDB).
* **Responsive Experience:** Fully optimized for all devices (Mobile, Tablet, Desktop).
* **Dynamic Theme:** Includes a Light/Dark mode toggle for personalized viewing.

##  Why the project is useful

* **First Impression:** High-end 3D graphics (Spline/Three.js) immediately grab the attention of recruiters and clients.
* **Performance:** Built with Vite for lightning-fast loading speeds and smooth scroll animations.
* **Clean Architecture:** Modular component structure makes it easy to update content or add new features.
* **Local Focus:** Tailored for the **Indore** freelance market, helping local businesses see the potential of modern web tech.

##  How to get started

### Prerequisites
* [Node.js](https://nodejs.org/) (v16.0.0 or higher)
* npm or yarn

### Installation

1.  **Clone the repository:**
    

    ``` 
    https://github.com/ramkachhwaha/3d-portfolio-website.git
    ```
    ```
    cd 3d-portfolio-website
    ```

2.  **Install dependencies:**
    ```
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser.

### Usage Example
To update your personal info, navigate to `src/constants/index.js` (or your data file) and modify the strings:
```javascript
export const navLinks = [
  { id: "about", title: "About" },
  { id: "work", title: "Work" },
  { id: "contact", title: "Contact" },
];
````
